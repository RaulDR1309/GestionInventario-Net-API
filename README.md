# 📦 Sistema de Gestión de Inventario - API REST

Esta es una API robusta desarrollada con **.NET 8/9** para la gestión de productos en un inventario. El proyecto demuestra el uso de una arquitectura profesional, conexión a bases de datos relacionales y documentación interactiva.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** C# 
* **Framework:** ASP.NET Core Web API
* **ORM:** Entity Framework Core (Code First)
* **Base de Datos:** SQL Server
* **Documentación:** Swagger / OpenAPI

## 🛠️ Características del Proyecto
* **CRUD Completo:** Implementación de métodos GET, POST, PUT y DELETE para el manejo de productos.
* **Persistencia de Datos:** Uso de migraciones para mantener la estructura de la base de datos sincronizada con el código.
* **Programación Asíncrona:** Mejora del rendimiento mediante el uso de `async` y `await`.
* **Inyección de Dependencias:** Gestión limpia del contexto de base de datos.
